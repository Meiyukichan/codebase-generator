# Codebase Generator — 大型项目文档树生成器

为大型代码库（50万行代码以上）生成分层结构化文档树。

**输出是中文。**

---

## 功能

根据源代码自动生成分层结构的项目文档，包含：
- 各级索引文件（toc.md）
- 详细的 spec 文档

---

## 输入

| 参数 | 说明 |
|------|------|
| **项目路径** | 待分析的源代码根目录 |
| **文档树路径** | 生成的文档树输出目录 |

---

## 文档树结构

```
{文档树路径}/
├── toc.md                          ← 总纲
├── {模块-a}/
│   ├── toc.md                      ← 模块索引
│   ├── {子模块-1}/
│   │   ├── toc.md                 ← 子模块索引
│   │   ├── spec-xxx.md
│   │   └── spec-yyy.md
│   └── spec-zzz.md                 ← 直接挂在模块下的spec
├── {模块-b}/
│   ├── toc.md
│   └── ...
└── ...
```

| 层级 | 说明 |
|------|------|
| **总纲** | 项目级索引 |
| **模块** | 大领域聚集（如工具模块、插件模块） |
| **子模块** | 小领域聚集 |
| **spec 文档** | 最小粒度，对应具体功能/接口的详细规格说明 |
